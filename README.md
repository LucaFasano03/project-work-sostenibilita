# Project Work – Sostenibilità Cantine PaoloLeo

Questo repository contiene una pagina web realizzata in HTML, CSS e JavaScript per la comunicazione delle attività di sostenibilità di un’impresa del settore vitivinicolo.

## Avvio del progetto
Aprire il file `index.html` tramite Live Server in Visual Studio Code. (Strumento utilizzato da me)

## Struttura del progetto
- `index.html` – struttura della pagina
- `css/` – foglio di stile
- `assets/` – immagini e favicon
- `reports/` – report di sostenibilità 2023 e 2024

## Fonti
- Report di sostenibilità Cantine PaoloLeo
- Sito ufficiale Cantine PaoloLeo (uso didattico)

## Funzionalità
La pagina utilizza JavaScript per gestire semplici interazioni, come il menu nella versione mobile.

## Note
I contenuti e le immagini sono utilizzati a scopo didattico e rielaborati a partire da fonti ufficiali.
